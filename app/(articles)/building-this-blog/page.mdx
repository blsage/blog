This site is a small Next.js app where every post is an MDX file that *is* its own route. There’s no CMS and no database — publishing means adding a folder with a `page.mdx` inside and one line to a JSON registry.

That registry drives the homepage index:

```ts
export function getPosts(): Post[] {
  return [...postsData.posts].sort((a, b) => b.date.localeCompare(a.date));
}
```
